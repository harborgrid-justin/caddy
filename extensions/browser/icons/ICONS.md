# Extension Icons

Place your extension icons in this directory with the following specifications:

## Required Icons

### Chrome/Edge
- `icon-16.png` - 16x16px (toolbar, tabs)
- `icon-48.png` - 48x48px (extension management)
- `icon-128.png` - 128x128px (Chrome Web Store, installation)

### Firefox
- `icon-48.png` - 48x48px (add-ons manager)
- `icon-96.png` - 96x96px (high-DPI displays)

## Design Guidelines

### Brand Colors
- Primary: `#667EEA` (Purple-blue gradient)
- Secondary: `#764BA2` (Deep purple)
- Accent: `#3B82F6` (Blue)

### Icon Design
- Use accessibility symbol (universal access icon)
- Include "A11y" or "CADDY" branding
- Ensure visibility on both light and dark backgrounds
- Follow platform-specific design guidelines

### File Format
- PNG with transparency
- 32-bit RGBA color
- Optimized for web
